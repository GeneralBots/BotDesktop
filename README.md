# BotDesktop

An AI-powered desktop automation tool that records and plays back user interactions.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create a .env file with your Azure OpenAI credentials

3. Development:
```bash
npm run dev
```

4. Build:
```bash
npm run build
```

## Testing
```bash
npm test
```
